Migrating from DeltaStyles?

    

      Host your skins on GitHub for free — faster updates, version control, and automatic catalog indexing.
    

    

      
        How to migrate in 5 minutes ›
      
      
    
        
  1. Create a free GitHub repo — e.g. yourname/skins
    2. 
        
  2. Upload your .deltaskin / .manicskin files — drag and drop in the GitHub web UI
    3. 
        
  3. Get the raw URL for each file: click the file → Raw → copy the URL
    4. 
        
  4. Submit the repo URL here — paste https://github.com/yourname/skins and click Fetch
    5. 
        
  5. Or register the repo for automatic weekly imports
    6. 
      

      

        GitHub gives you unlimited public repo storage, version history, and the raw URL never changes
        (as long as the branch/path stays the same). You can even use GitHub Releases to distribute
        versioned skin bundles.
